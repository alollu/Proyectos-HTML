# 🍽️ Weekly Menu Generator

Generador de menús semanales saludables con control de proteínas. Crea dietas equilibradas evitando alergenos comunes como lácteos, mariscos y tofu.

## ✨ Características

- 🎲 Generación automática de menús semanales
- 🔀 Intercambio de platos entre días
- 📊 Control de ingesta proteica (100-120g/día)
- 🚫 Excluye alérgenos comunes
- 📱 Interfaz responsive y intuitive
- 💾 Funciona offline

## 🚀 Cómo usar

1. Haz clic en "Generar Nuevo Menú" para crear una semana
2. Usa "Mezclar Platos" para redistribuir las comidas
3. Verifica que las proteínas estén en rango (100-120g/día)
4. Personaliza las comidas según tus preferencias

## 🛠️ Tecnologías

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript Vanilla

## 📦 Instalación

Descarga el archivo HTML y ábrelo directamente en tu navegador. No requiere instalación.

## 🥗 Comidas incluidas

- **Desayuno**: Opciones ricas en proteínas
- **Comida**: Platos principales equilibrados
- **Tentempié**: Snacks saludables
- **Cena**: Cenas ligeras y nutritivas

## ⚠️ Consideraciones

El menú excluye intencionalmente:
- Lácteos
- Mariscos y gambas
- Tofu
- Quesos azules

Puedes modificar el array `suggestions` en el código para añadir o personalizar platos.
